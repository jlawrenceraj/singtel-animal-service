# singtel-animal-service

Run the spring boot application

#Rest Endpoints (running in port 9080)
- GET `/animals` - This endpoint will return all the animal types
- GET `/animals/{activity}` - This endpoint will return all animal types who can do the given activity type
- GET `/animals/{activity}/count` This endpoint will return the count of animals who can do the given activity type
